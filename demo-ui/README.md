# Demo UI

Bu klasor, ExamFlow projesi icin demo amacli hafif arayuz calismalarini barindirir.

Ilk hedef:

- `/publish` akisini gorsel olarak tetiklemek
- `/health` ve `/ready` durumlarini gostermek
- backend response bilgisini ekranda sunmak

Not:

- Bu klasor su an sadece iskelet amaciyla olusturuldu.
- Ayrica tam frontend projesi kurulumu daha sonra bu klasor icinde yapilacaktir.
